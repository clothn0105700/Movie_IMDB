# Movie_IMDB
串接 The Movie Database (TMDB) API 抓取 IMDB(影音評分網站)資料  
並實作篩選和搜尋電影功能

# 使用套件
React Router : 處理路由  
axois : 代替Fetch抓取API  
swiper : 製作電影輪播圖  
react-spinners : 物件loading特效  

# 如何在本地端執行

安裝
```shell
$ npm install
```
執行
```shell
$ npm dev run
```
